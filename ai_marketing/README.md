# AI Marketing & Creative Automation

This folder contains AI-assisted creative workflows, advertising systems, and content generation pipelines I developed for marketing, branding, and promotional operations.

The focus of these projects is rapid creative iteration, scalable asset generation, multi-format campaign production, and workflow acceleration using modern AI tools combined with traditional design and automation techniques.

Rather than treating AI as a one-click generator, these workflows were designed as structured production systems with repeatable outputs, prompt engineering, formatting pipelines, and cross-platform deployment considerations.
