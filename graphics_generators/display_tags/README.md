# 🏷️ Display Tag Generator

## Overview
The **Display Tag Generator** creates universal display tags for any product type — flower, edibles, cartridges, topicals, etc. Just scan or enter products from around the store that need updated shelf tags.

## How to Use
1. Scan or type a **SKU** into **Column A**.
2. The sheet autofills all product data from your internal lookup tables.
3. Go to `Extensions → Autocrat`.
4. Run the **Display Tag Generator** merge.

## Output
- PDF tags are generated and saved to your designated Google Drive folder.

## Requirements
- Google Sheets
- [Autocrat Add-on](https://workspace.google.com/marketplace/app/autocrat/539235305518)
