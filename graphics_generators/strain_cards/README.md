# 🌿 Strain Card Generator

## Overview
The **Strain Card Generator** produces visually branded strain cards for flower products on display in your store. It uses SKU-based autofill and custom descriptors to create a professional-looking PDF for each strain.

## How to Use
1. Paste or scan a **SKU** into **Column A**.
2. Wait for the sheet to autofill the product data using internal lookup formulas.
3. Fill in the **highlighted fields**:
   - THC %
   - CBD %
   - Total Cannabinoids
4. Select **two buzzword descriptors** from the adjacent dropdowns (e.g., "Uplifting", "Euphoric").
5. Go to `Extensions → Autocrat`.
6. Run the **Strain Card Generator** merge.

## Output
- A PDF is generated for each completed row.
- Files are saved in your connected Google Drive folder.

## Requirements
- Google Sheets
- [Autocrat Add-on](https://workspace.google.com/marketplace/app/autocrat/539235305518)

## Notes
- Make sure all required fields are filled before running the merge.
- Only edit rows that need new cards.
