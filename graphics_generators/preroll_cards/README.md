# 🚬 Preroll Card Generator

## Overview
The **Preroll Card Generator** creates simple shelf cards or tray cards for preroll products using SKUs pulled from your POS data.

## How to Use
1. Paste or scan a **SKU** into **Column A**.
2. The product name, brand, size, and other data will autofill automatically.
3. Open `Extensions → Autocrat`.
4. Run the **Preroll Card Generator** merge.

## Output
- One PDF per preroll entry is created and stored in your Google Drive.

## Requirements
- Google Sheets
- [Autocrat Add-on](https://workspace.google.com/marketplace/app/autocrat/539235305518)

## Notes
- This tool requires no manual entry besides scanning the SKU.
- Ideal for batch processing large preroll inventories.
